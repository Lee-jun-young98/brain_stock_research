# Brain_stock_research

## Purpose
- 데이터셋을 이용하여 뇌졸중을 가장 잘 예측할 수 있는 모델 만들기

## Data
- https://www.kaggle.com/fedesoriano/stroke-prediction-dataset

## Feature analysis
- 수치형 변수들 간의 상관분석 시행

## PreProcessing
- bmi 결측치 -> 평균값으로 대체
- 라벨인코딩, 변수 스케일링 사용
- SMOTE 알고리즘을 이용하여 label 불균형 해결

## Model
- Logistic Regression
- SVM
- Decision Tree
- Random Forest
- Gaussian NB
